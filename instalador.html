Preparar unidad USB como instalador

Descargar Windows 11 AQUÍ
Descargar Windows 11 AQUÍ (2)

Para preparar una unidad USB como instalador de Windows 11, sigue estos pasos:

  1. Descarga la herramienta de creación de medios de Windows 11 desde el sitio oficial de Microsoft.
  2. Conecta una unidad USB a tu computadora (asegúrate de que tenga al menos 8 GB de espacio).
  3. Ejecuta la herramienta de creación de medios y selecciona "Crear medio de instalación (unidad flash USB, DVD o archivo ISO)".
  4. Selecciona el idioma, la edición y la arquitectura (32 bits o 64 bits) que deseas para Windows 11.
  5. Selecciona "Unidad flash USB" como medio a utilizar.
  6. Elige la unidad USB que deseas usar y sigue las instrucciones para crear el medio de instalación.

Instalación de Windows 11

Para instalar Windows 11, sigue estos pasos:

  1. Conecta la unidad USB que preparaste anteriormente a tu computadora.
  2. Reinicia tu computadora y accede al menú de arranque (F2).
  3. Es posible que al no tener Sistema Operativo, vaya directamente al menú de arranque
  4. Selecciona la unidad USB como dispositivo de arranque/boot => Si hay una lista, mover a primer lugar (los controles suelen aparecer en pantalla, ¡aunque el tuyo igual tiene raton en esta pantalla!, pero no es lo normal)
  5. Guardar y salir => Se reiniciará y ahora cargara el USB de instalación
  6. Sigue las instrucciones en pantalla para instalar Windows 11 (nombre, correo,...)
  7. Cuando llegues a la pantalla de WIFI o de instalar Drivers, tienes que darle datos con el movil mediante un cable (que tú pc todavia no sabe que tiene wifi/bluethoot, pa eso son los drivers, para que sepa que tiene y como se usa)
